# Battleboard 3.0 🔥

你的人生游戏系统 - 将真实生活转换成游戏机制的"人生驾驶舱"

## 快速开始

### 方法一：直接打开（最简单）
1. 双击 `index.html` 用浏览器打开
2. 开始使用！

### 方法二：添加到手机主屏幕（推荐）
1. 用手机浏览器打开 `index.html`（可以通过本地服务器或部署到网上）
2. **iOS**: Safari → 分享按钮 → "添加到主屏幕"
3. **Android**: Chrome → 菜单 → "添加到主屏幕"

### 方法三：本地服务器
```bash
# 使用Python
python -m http.server 8000

# 或使用Node.js
npx serve
```
然后访问 http://localhost:8000

## 核心功能

### 🌱 基础努力（绿色）
- 最低要求：任意一个emoji
- 吃得健康、散步、做拉伸、整理房间...
- **每天至少1个emoji = streak继续**

### 📘 支线任务（蓝色）
- 更有建设性的活动
- 学习、写作、规划、心理建设...

### 🔥 主线任务（红色）
- 核心深度工作
- 完成晚间3问
- **通关 = +10 XP**

### 📊 等级系统
- Lv.1 → Lv.2 需要 100 XP
- 每升一级，XP需求增加100
- 可视化经验条

### 🎁 每周抽奖
- 连续7天streak → 解锁宝箱
- 随机抽取奖励池中的奖励
- 可自定义奖励内容

### 📅 周历视图
- 每天显示优先级最高的2个emoji
- 主线日会特别高亮

## 数据说明

所有数据保存在浏览器的 localStorage 中，包括：
- streak 记录
- XP 和等级
- 每日emoji
- 晚间3问内容
- 奖励池

### 备份数据
设置 → 导出数据 → 保存JSON文件

### 恢复数据
设置 → 导入数据 → 选择JSON文件

## 部署到网上（可选）

### GitHub Pages（免费）
1. 创建GitHub仓库
2. 上传所有文件
3. Settings → Pages → 选择main分支
4. 访问 `https://你的用户名.github.io/仓库名`

### Vercel/Netlify（免费）
1. 连接GitHub仓库
2. 自动部署

## 自定义图标

打开 `generate-icons.html` 生成应用图标，然后保存为：
- `icon-192.png`
- `icon-512.png`

## 技术栈

- 纯HTML/CSS/JavaScript
- localStorage 本地存储
- PWA 支持离线使用
- 无需后端服务器
